# Agentic RAG Application - Assignment 7

This repository contains an advanced Retrieval-Augmented Generation (RAG) agent that utilizes the Model Context Protocol (MCP) to seamlessly fetch, index, and query external knowledge sources. 

The agent uses:
- `fetch_url` to retrieve documents from the web.
- `index_document` to chunk and embed documents into a persistent FAISS vector database.
- `search_knowledge` to retrieve semantically relevant information from the knowledge base to answer complex reasoning tasks.

## Running the Evaluation Suite

To run the automated tests consisting of 8 base queries and 5 custom RAG queries (tested both with and without the FAISS index), simply execute:

```bash
bash run_queries.sh
```
